Все .img (USB Memstick образы) могут быть записаны на USB носитель через команду: 
dd if=_path_to_img.img_ of=/dev/da0 bs="10240" conv="sync"
Будьте внимательны, прописывая корректное имя устройства в of= аргументе.

В данный момент, генерация cbsd образов отключена. Для установки из исходников воспользуйтесь этой инструкцией
Если же все равно что-то непременно хотите скачать ;-), скачайте FreeBSD LiveCD из rescue jail: Rescuecbsd LiveCD

pkg репозитории из которых собраны клетки (для /usr/local/etc/pkg.conf):

FreeBSD 9.2
PACKAGESITE:http://dl.bsdstore.ru/freebsd/amd64/9.2/xorg # (обычная версия, NVIDIA GL)
PACKAGESITE:http://dl.bsdstore.ru/freebsd/amd64/9.2/xorgkms # (Xorg with KMS)
PACKAGESITE:http://dl.bsdstore.ru/freebsd/amd64/9.2/nox # (WITHOUT_X11=yes, php55, MariaDB)
PACKAGESITE:http://dl.bsdstore.ru/freebsd/amd64/9.2/nox11 # (WITHOUT_X11=yes)
FreeBSD CURRENT aka 10.0
PACKAGESITE:http://dl.bsdstore.ru/freebsd/amd64/10.0/xorg # (обычная версия, NVIDIA GL)
PACKAGESITE:http://dl.bsdstore.ru/freebsd/amd64/10.0/xorgkms # (Xorg with KMS)
PACKAGESITE:http://dl.bsdstore.ru/freebsd/amd64/10.0/nox # (WITHOUT_X11=yes, php55, MariaDB)
PACKAGESITE:http://dl.bsdstore.ru/freebsd/amd64/10.0/nox11 # (WITHOUT_X11=yes)

Прим1: в данный момент сборка выполняется только для x86-64 архитектуры пр причине отсутствия в моей ферме иных.
Прим2: данные имиджи и репозитории подняты и поддерживаются больше в целях демонстрации и проверок функционала cbsd, нежели для обеспечения чьей-то работы. В целях безопасности, приучайте себя не доверять любым репозиторям ПО в интернете, используйте только официальные и хорошо проверенные репозитории либо собирайте собственные.
