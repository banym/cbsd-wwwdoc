Синтаксис команд cbsd


Все команды начинаются с префикса cbsd: 

% cbsd jls
% cbsd jstop jname
% cbsd jstart jname
% cbsd node mode=list
...

Команды cbsd могут принимать аргументы, записываемые в виде "param=value"
cbsd команды могут иметь как обязательные аргументы, так и аргументы, которые можно опустить. Получить список аргументов можно по команде

% cbsd <cmd> --help
- с выводом и кратким описанием, какие команды обязательны, а какие опциональны.

либо

% cbsd <cmd> --args
для вывода всех ожидаемых аргументов у команды cmd.

Пример:

Команда 
% cbsd --help
выведет список всех возможных команд cbsd.

Команда 
% cbsd jsnapshot --help
выведет все возможные параметры для команды jsnapshot
