Rescuebsd LiveCD

rescuebsd и xrescuebsd - это клетки для cbsd (консольная и с X сервером соответственно), содержащие популярный набор ПО для настройки и восстановления систем, утилиты для анализа сети и тестирования аппаратного обеспечения, которые могут быть полезны в качестве дистрибутивов при восстановительных работах. Работа над разработкой LiveCD дистрибутива в мои планы не входит, нижедоступные образы являются по большей части побочным эффектом от jail2iso и демонстрацией ее работы.

Данные клетки могут быть получены через cbsd repo, кастомизированы, и использованы в качестве PXE -клеток для бездисковой загрузки по сети, либо посредством cbsd jail2iso получены USB Memstick и DVD/CD образы.

Список ПО, входящего в оригинальные клетки: rescuebsd, xrescuebsd

Все .img (USB Memstick образы) могут быть записаны на USB носитель через команду: 
dd if=_path_to_img.img_ of=/dev/da0 bs="10240" conv="sync"
Будьте внимательны, прописывая корректное имя устройства в of= аргументе.
