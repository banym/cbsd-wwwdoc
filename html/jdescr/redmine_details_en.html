Redmine jail



To install redmine cbsd jail execute the following commands:

cbsd repo action=get sources=img name=redmine
After downloading and set correct ip address of jail, run install script who will ask you for a password for root user in MySQL, password of redmine user in MySQL and the name of the resource will be available via http://FQDN

If all goes well, when you start the jail, you automatically get a clean Redmine installation which is available for configuring via Redmine default login and password: admin
