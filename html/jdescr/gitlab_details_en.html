Gitlab jail

To install gitlab cbsd jail execute the following commands: 
% cbsd repo action=get sources=img name=gitlab
After downloading images and set correct ip address of jail, run install script who will ask you for a password for root user in MySQL, password of gitlab user in MySQL and the name of the resource will be available via http://FQDN: 
% cbsd jconfig jname=gitlab gitlabinstall

If all goes well, when you start the jail: 

% cbsd jstart gitlab
your node automatically get a clean Gitlab installation in the jail which is available for configuring via http://FQDN with Gitlab default login and password: admin
