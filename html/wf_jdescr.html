Команды:
% cbsd jdescr
Описание:

Для каждой клетки можно ввести какое-то описание/аннотацию. cbsd jdescr без параметров выводит описание для всех клеток на ноде.
Команда
% cbsd jdescr mode=update jname=jname
запустит редактор vi для редактирования описания. Через параметр editor можно указать альтернативный редактор (например, понимающий UTF8) - vim, mcedit и тп. Описание хранится в ASCII-файле, сохраняемый в каталоге $workdir/jails-system/jname/descr, где jname - имя редактируемой клетки. Соответственно, файл можно просто скопировать. Поскольку каталог jails-system учавствует в jcoldmigrate, jclone, jimport/export и тп операциях, описание будет сохраняться.

Пример (запустить редактор mcedit для редактирования описания клетки kde4): 
% cbsd jdescr jname=kde4 editor=mcedit mode=update
