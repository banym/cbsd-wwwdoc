Команды:
% cbsd jclone

% cbsd jrclone
Описание:

Выполняет клонирование jail в новый. В качестве обязательных аргументов, указывается источник/оригинал клетки через old,новое имя через аргумент new и FQDN (hostname) через fqdn. В качестве необязательного - новые ip адреса через ip4_addr (если IP несколько, адреса указываются через запятую без пробелов)

Пример (клонирование клетки amp123 в amp123clone со сменой ip, fqdn): 
% cbsd jclone old=amp123 new=amp123clone fqdn=amp123clone.my.domain ip4_addr=10.0.0.200/24
