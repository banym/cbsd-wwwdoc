commands:
% cbsd history
Descriptions:

When cbsd command comes from the terminal and no set the environment variable NO_CBSD_HISTORY, then the command will be logged in the $workdir/.cbsd_history file.

You can use the command 
  % cbsd history [N]
(where N - optional parameter) to display all entered cbsd commands or only N last.

File size $workdir/.cbsd_history by default limited in 300 Kb and when limit is reached the oldest records will be deleted.
