Команды:
			% cbsd node mode=list

Команда cbsd node mode=list выведет имена тех нод, RSA ключи которых были получены через cbsd node add. Тоже самое можно получить, просмотрев каталог $workdir/.rssh, в котором находятся ключи. Выведенный список не означает, что имеющийся ключ актуален или связь с нодой имеется (для этих задач есть специальный демон nodepinger)

Пример:

			% cbsd node mode=list
