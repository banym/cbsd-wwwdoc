Скачать

Все .img (USB Memstick образы) могут быть записаны на USB носитель через команду:

	    % dd if=_path_to_img.img_ of=/dev/da0 bs="10240" conv="sync"

Будьте внимательны, прописывая корректное имя устройства в of= аргументе.

В данный момент, генерация CBSD образов отключена. Для установки из исходников воспользуйтесь этой инструкцией

Если же все равно что-то непременно хотите скачать ;-), скачайте FreeBSD LiveCD из rescue jail: Rescuecbsd LiveCD

Список преднастроенных имиджей

Ниже приведен список преднастроенных имиджей, доступные для скачивания через команду cbsd repo action=get sources=img name=name. Данные образы могут представлять из себя «кубики» из которых можно собирать необходимую вам инфраструктуру. Вы ограничены только физическими ресурсами сервера, по исчерпанию которых всегда можно поставить еще одну ноду и мигрировать часть клеток на нее. Каждый образ представляет из себя архив клетки с необходимым ПО и имеющим минимальные шаблоны конфигурационных файлов, на которые отрабатывает хелпер применяющий ввод первоначальных пользовательских данных (например пароль администратора приложения) для того, чтобы приложение заработало как можно быстрее. Далее работать с клеткой, сопровождать конфигурацию и ПО можно стандартным способом — она независима от CBSD. Все хелперы могут отрабатывать неинтерактивно (все возможные аргументы доступны по команде --help), являясь примером массового построения и разворачивания Application серверов из клеток в автоматическом режиме.
