Drupal/MySQL jail

For installing Drupal7 with MySQL backend just follow this steps (for FreeBSD 10 and above):

			% cbsd repo action=get sources=img name=drupal

or

			% cbsd repo action=get sources=img name=drupal ver=10.0

If you hosted on another ( non 10.0 ) FreeBSD version. After image downloading please set correct name and ip address of the jail and run a install script that asks you for FQDN of WEB server and Drupal:

			% cbsd jconfig jname=drupal install

After entering FQDN of instance, Install procedure and starting jail the clean Drupal installation will be available via http://FQDN/
