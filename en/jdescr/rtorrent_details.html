Rtorrent for FreeBSD CBSD Jail

Rtorrent jail

To install rtorrent cbsd jail execute the following commands:

			% cbsd repo action=get sources=img name=rtorrent

After downloading images and set correct ip address of jail, run install script who will ask you for a password for rtorrent rpc user and server FQDN. After that rutorrent WEB interface will be available via http://FQDN/rtorrent

			% cbsd jconfig jname=rtorrent install

If all goes well, when you start the jail:

			% cbsd jstart rtorrent
