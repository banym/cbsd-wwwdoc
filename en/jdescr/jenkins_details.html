Jenkins FreeBSD jail image

Jenkins is a free extendable open source continuous integration server

For installing Jenkins CBSD jail just follow this steps:

			% cbsd repo action=get sources=img name=jenkins

or

			% cbsd repo action=get sources=img name=jenkins ver=10.0

If you hosted on another ( non 10.0 ) FreeBSD version. After image downloading please set correct name and ip address of the jail

A clean Jenkins instance in the jail will be available for access via http://IP:8180/jenkins/
