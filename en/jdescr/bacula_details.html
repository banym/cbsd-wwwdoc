Bacula jail

To install bacula cbsd jail execute the following commands: 
% cbsd repo action=get sources=img name=bacula
After downloading images and set correct ip address of jail, run install script who will ask you for a password for root user in MySQL, password of bacula user in MySQL and the name of the resource will be available via http://FQDN: 
% cbsd jconfig jname=bacula baculainstall

If all goes well, when you start the jail: 

% cbsd jstart bacula
