History of CBSD commands

history command

			% cbsd history

Descriptions:

When CBSD command comes from the terminal and no set the environment variable NO_CBSD_HISTORY, then the command will be logged in the $workdir/.cbsd_history file.

You can use the command

		% cbsd history [N]

(where N â€” optional parameter) to display all entered CBSD commands or only N last. File size $workdir/.cbsd_history by default limited in 300 Kb and when limit is reached the oldest records will be deleted.
